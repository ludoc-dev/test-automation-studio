# Roadmap: Test Automation do Zero ao Deploy

## Fase 1: Fundamentos BDD (2-3 semanas)

### Semana 1: Behave + Gherkin
- [ ] Syntax básica do Gherkin (Feature, Scenario, Given/When/Then)
- [ ] Estrutura de diretórios do Behave
- [ ] Steps definitions em Python
- [ ] Contexto e partilha de dados entre steps

### Semana 2: Selenium WebDriver
- [ ] Configuração do WebDriver (Chrome/Firefox)
- [ ] Locators strategies (ID, CSS, XPath)
- [ ] Esperas explícitas vs implícitas
- [ ] Interação com elementos (click, send_keys, select)

### Semana 3: Page Object Model
- [ ] Design Pattern POM
- [ ] Separação de lógica de navegação
- [ ] Reutilização de componentes
- [ ] Factory pattern para pages

## Fase 2: Testes de Carga (1-2 semanas)

### Semana 4: Locust Fundamentals
- [ ] Instalação e configuração
- [ ] HttpUser e tasks
- [ ] wait_time e between
- [ ] weight e priority

### Semana 5: Carga Avançada
- [ ] Distributed load (master/worker)
- [ ] Cenários complexos
- [ ] Extração de dados dinâmicos
- [ ] Métricas e thresholds

## Fase 3: CI/CD com GitLab (1-2 semanas)

### Semana 6: Pipeline Básico
- [ ] .gitlab-ci.yml structure
- [ ] Stages (test, build, deploy)
- [ ] Jobs e scripts
- [ ] Artifacts e reports

### Semana 7: Integração Completa
- [ ] Test execution em CI
- [ ] Parallel execution
- [ ] Quality gates
- [ ] Deploy automation
