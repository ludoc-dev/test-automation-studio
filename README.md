# Test Automation Studio

Ambiente completo para desenvolvimento de testes automatizados - de BDD até testes de carga e CI/CD.

## 🎯 Roadmap de Aprendizado

### Fase 1: Fundamentos BDD
- [x] Behave (Python BDD framework)
- [x] Gherkin (syntax de feature files)
- [x] Page Object Model (design pattern)
- [x] Selenium WebDriver

### Fase 2: Testes de Carga
- [ ] Locust (testes de carga distribuídos)
- [ ] Cenários de escalabilidade
- [ ] Métricas e monitoramento

### Fase 3: CI/CD
- [ ] GitLab CI/CD pipelines
- [ ] Integração contínua
- [ ] Deploy automatizado

## 📂 Estrutura

```
test-automation-studio/
├── frameworks/         # Implementações de exemplo
├── docs/              # Documentação de estudo
├── projects/          # Projetos práticos
├── scripts/           # Scripts utilitários
└── skills/            # Skills para automação
```

## 🚀 Quick Start

```bash
cd ~/test-automation-studio

# Criar novo projeto
./scripts/new-project.sh meu-projeto

# Limpar traços de testes
./skills/cleanup.sh
```
